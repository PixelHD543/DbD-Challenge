# Bead by Daylight 🎯🍻

A random challenge generator for Dead by Daylight with drinking game mode!

## Features
- 🎲 Random survivor/killer challenges
- 🎭 Random perk builds (4 perks + items)
- 🍺 Drinking mode with custom rules
- 📜 Challenge history
- 🎯 Specific objectives for each match

## How to Use
1. Open `index.html` in any modern browser
2. Select NORMAL or DRINKING mode
3. Click GENERATE CHALLENGE
4. Play Dead by Daylight following your challenge!
5. Press SPACE bar for quick generation

## Drinking Mode Rules
- Normal mode: Sip = small drink
- Hardcore mode: Shot = full drink
- Adjust to your tolerance!

## Customization
Edit `script.js` to add:
- More perks
- Custom challenges
- New drinking rules
- Different difficulty levels

## Deployment to GitHub
1. Create a new repository
2. Upload all 4 files
3. Enable GitHub Pages in Settings
4. Your page will be live at `username.github.io/repo-name`
